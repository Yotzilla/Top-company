Local AI (Ollama)
ฟรี 100% บนเครื่องคุณ
วิธีเปิด Ollama ให้ Top Company เข้าถึงได้:
1. ติดตั้ง Ollama → ollama.com/download
2. PowerShell: $env:OLLAMA_HOST="0.0.0.0:11434"
3. PowerShell: ollama serve
4. ติดตั้ง cloudflared → winget install Cloudflare.cloudflared
5. cloudflared tunnel login
6. cloudflared tunnel create top-ai
7. cloudflared tunnel run --url http://localhost:11434 top-ai
8. Copy URL → ใส่ใน Worker env OLLAMA_URL
🔐 ความปลอดภัย — API Key เก็บอยู่ใน Cloudflare Worker เท่านั้น ไม่มีใน HTML ไม่โผล่ใน source code และไม่ถูกส่งมายัง browser เลย
